# Pick One

This directory contains prebuilt KirOpen outputs for the supported harnesses.

Choose one mode:

- `agent/`
  Use this when you want KirOpen to stay mostly opt-in through harness-specific agents, shared skills, prompts, and scoped guidance.
- `default/`
  Use this when you want KirOpen to also shape the harness's default repository-wide behavior.

Both folders contain prebuilt outputs for the currently supported harnesses:

- Codex
- GitHub Copilot

Copy the files you want from the chosen mode into the target repository, following the installation guidance in this repo's `README.md`.
